## 📌 Sobre o projeto

Projeto desenvolvido para **fins de estudo**, focado na construção de interfaces de autenticação com HTML e CSS puro — sem frameworks, sem JavaScript complexo.

Conta com duas telas funcionais e navegação entre elas.

## ✨ Funcionalidades

| | Tela |
|---|---|
| 🔑 | **Página de Login** — campos de usuário e senha com botão de entrar |
| 📝 | **Página de Criar Conta** — nome, sobrenome, e-mail, criar e confirmar senha |
| 🔗 | Navegação entre as páginas via links |

---

## 📁 Estrutura do projeto

```
login-pag/
├── index.html       # Página de login
├── NewA.html        # Página de criar conta
├── css/
│   ├── front.css    # Estilo da página de login
│   └── NewA.css     # Estilo da página de criar conta
└── imagens/
    └── Group_38 2.png  # Logo (substituir se for usar)
```

---

## 🚀 Como usar

1. Clone o repositório:

```bash
git clone https://github.com/SamuelG05/login-pag.git
```

2. Abra o arquivo `index.html` no navegador.

> **Atenção:** Se for reutilizar o projeto, substitua a imagem de logo na pasta `imagens/`.
